# MVP Routes

## Route A: Fast Article Pipeline
1. Collect source
2. Convert to markdown
3. Summarize and extract key points
4. Generate one article draft
5. Generate one short social version
6. Save deposition

## Route B: WeChat-Centered Pipeline
1. Collect source
2. Produce article markdown
3. Generate cover / illustrations
4. Prepare WeChat draft
5. Save deposition and reusable fragments

## Route C: One Source, Many Outputs
1. Normalize source
2. Extract central thesis
3. Generate long-form article
4. Generate social short-post
5. Generate slide outline or image prompt
6. Deposit assets for reuse
